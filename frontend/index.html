🚀 Ajouter un personnage EVE

Cliquez sur le bouton ci-dessous pour vous connecter avec votre compte EVE Online.
Vous serez redirigé vers le site EVE pour autoriser l'accès à vos BPO.
Aucune configuration nécessaire — l'application est déjà enregistrée.

Pour connecter votre personnage, vous devez enregistrer une application sur le portail développeur EVE Online. Suivez les étapes ci-dessous :

  1. Allez sur developers.eveonline.com et connectez-vous (ou créez un compte)
  2. Cliquez Manage Applications → Create New Application
  3. Remplissez :
    • Name : ce que vous voulez (ex: BPO Dashboard)
    • Description : optionnel
    • Connection Type : Authentication & API Access
    • Permissions : ajoutez le scope esi-industry.read_character.blueprints.v1
    • Callback URL : copiez celui indiqué ci-dessous
  4. Validez, puis copiez le Client ID et Client Secret ci-dessous

http://localhost:8090/api/sso/callback
